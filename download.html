PBN Conversion Utility Program

This utility program will convert PBN files into a format that can be used by Bridge Captain 6. You tell the program the starting folder and it will:

  1. Find all PBN files in that folder, and all sub-folders
  2. Convert each file so that it will be readable by BC 6
  3. Copy the converted file(s) to the "Bridge Captain PBN Files" folder which will be created on your Desktop.
