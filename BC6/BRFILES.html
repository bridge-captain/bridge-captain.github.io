Selecting Multiple Files

Top  Previous  Next

 

Bridge Captain allows you to select multiple files.  This is handy when you want to delete several files. Hold down the Ctrl key while you select all the files to be deleted. Then you can click the Delete Files(s) button to transfer the selected files into the Recycle Bin. You must also hold down the Ctrl key if you want to select a PBN file to be deleted.

 

Finding Bridge Files

Top  Previous  Next

 

Use the Select Folder button to find the folder for the desired Bridge File.  All Bridge Files in that folder will be shown in the left column (under "Bridge Files"). When you click on a .PBN file, if the corresponding .INI file:

1. does not exist, Bridge Captain will create an .INI file with the same information; or

2. exists, Bridge Captain will ask if want to send the existing .INI file into the Recycle Bin and then create a new .INI file.

 

Bridge Captain saves the drive and folder from the last time you exited the Select Bridge File dialog box.  This location is then used as the starting location when you next enter this dialog box.

 

Bridge Files

Top  Previous  Next

 

Bridge Captain can read information about bridge hands that have been created using either the .INI or the .PBN file format. The file extension designates the format being used.:

.INI

used by many programs, especially in Windows

.PBN

designed specifically for transferring bridge hand data between computers, regardless of the operating system

 

Bridge Captain writes information about a hand only using the .INI file format.  Whenever you wish to look at and/or play a hand from a file with a .PBN format, Bridge Captain will automatically create an .INI file with the same information.

 

In summary. Bridge Captain creates and updates .INI files but also allows you to read and play.PBN files.

 

Selecting a board to be played

Top  Previous  Next

 

After you've selected a Bridge File, you have four options:

 

1. Delete the file, to send this file to the Recycle Bin.

2. Convert a PBN file to .INI format.

3. Play the first board that you haven't played.

 

If you want to play a board other than the first one you haven't played, click on the desired board number in the right hand column and then click

 

4. Play Selected Board

 

If you wish to delete several Bridge Files, you may hold down the Ctrl key while you select all the files to be deleted. Then you can click the Delete Files(s) button to transfer the selected files into the Recycle Bin.

 

NOTE: If you want to delete a .PBN file, you must hold down the Ctrl key while selecting it.

 

Network Drives

Top  Previous 

 

In order to use Bridge Files that are located on another computer, that computer must be located within your "Network Neighborhood" under Win98 or in "My Network Places" under WinXP.  Use the Select Folder button to navigate to the desired folder in your network.

 

Of course, the other computer in your network must be on and connected in order to play hands from a file on that computer.
